# <a name="partner-center-for-microsoft-cloud-germany"></a>Centro per i partner per Microsoft Cloud Germania

**Si applica a**

-  Centro per i partner per Microsoft Cloud Germania

## <a name="microsoft-cloud-germany-meeting-german-data-protection-requirements"></a>Microsoft Cloud Germania: conformità ai requisiti di protezione dei dati tedeschi 

Microsoft Cloud Germania è progettata per soddisfare i rigidi requisiti tedeschi per la protezione dei dati e include un trustee per i dati tedesco che controlla l'accesso fisico e logico ai dati dei clienti in base alle normative tedesche. Tutti i dati dei clienti e i sistemi che elaborano o controllano l'accesso ai dati dei clienti, si trovano nei centri dati tedeschi. La rete tra questi centri dati è isolata e dedicata all'interno della Germania.

Microsoft Cloud Germania offre un'istanza separata di [Microsoft Azure](https://go.microsoft.com/fwlink/?linkid=847992), Office 365 e Microsoft Dynamics, fornita dai centri dati in Germania. Di conseguenza, i clienti commerciali nelle aree UE ed EFTA possono archiviare e gestire i dati dei clienti in conformità alle leggi e alle normative tedesche nonché ai principali standard internazionali. Il personale Microsoft non può accedere a dati dei clienti senza l'approvazione e la supervisione da parte del trustee dei dati tedesco o del cliente.

Per ulteriori informazioni, vedere [la scheda tecnica per Microsoft Cloud Germania](http://download.microsoft.com/download/6/1/3/613C9ECB-9167-4EF5-B131-3BAD8D8A126C/Microsoft_Cloud_Germany_Datasheet.pdf).

## <a name="partner-center-for-microsoft-cloud-germany-selling-german-cloud-solutions-in-eu-and-efta"></a>Centro per i partner per Microsoft Cloud Germania: vendita di soluzioni cloud tedesche nelle aree UE ed EFTA

Dopo essersi registrati in CSP (Cloud Solution Provider) per il programma Microsoft Cloud Germania, si utilizzerà il Centro per i partner per vendere soluzioni e servizi tedeschi specifici per il cloud ai clienti nei paesi e nelle aree geografiche UE ed EFTA.

## <a name="frequently-asked-questions-about-the-csp-program-for-microsoft-cloud-germany"></a>Domande frequenti sul Programma CSP per Microsoft Cloud Germania

Utilizza queste domande frequenti per trovare ulteriori informazioni sul programma Cloud Solution Provider (CSP) per Microsoft Cloud Germania. Per ulteriori informazioni, puoi inoltre contattare il tuo account manager. 

**Che cos'è Microsoft Cloud Germania?** 

Microsoft Cloud Germania combina un'istanza di servizi cloud all'avanguardia di Microsoft con un'architettura di gestione dei sistemi unica e una rete dedicata all'interno del territorio tedesco. Con questo modello unico, il Trustee dei dati, una società indipendente tedesca, controlla l'accesso ai dati dei clienti e non li condivide con terze parti, ad eccezione di come indicato dal cliente o richiesto dalle normative tedesche. Questo concetto è detto anche Sovranità dei dati. 

**Chi è idoneo per il Programma CSP per Microsoft Cloud Germania?**

I partner nell'Unione Europea (UE), in EFTA e nel Regno Unito sono idonei per il Programma CSP per Microsoft Cloud Germania e possono vendere a clienti in UE, EFTA e Regno Unito. 

**CSP Commercial è un programma per partner esistente. In che modo si differenzia il Programma CSP per Microsoft Cloud Germania?**

Il Programma CSP per Microsoft Cloud Germania è diverso da CSP Commercial nelle aree seguenti: 
-   Idoneità: il Programma CSP per Microsoft Cloud Germania è disponibile solo per i partner in UE, EFTA e Regno Unito. 
-   Onboarding: i partner devono eseguire l'onboarding al Programma CSP per Microsoft Cloud Germania separatamente da CSP Commercial. 
-   Gestione delle licenze: i documenti di gestione delle licenze per il Programma CSP per Microsoft Cloud Germania sono stati aggiornati per riflettere il ruolo del Trustee dei dati. Ulteriori informazioni sulle modifiche alle licenze sono riportate di seguito. 
-   Funzionalità Centro per i partner: determinate funzionalità del Centro per i Partner non saranno abilitate per il Programma CSP per Microsoft Cloud Germania per soddisfare i requisiti relativi alla sovranità dei dati. I partner possono utilizzare invece Microsoft Online Portal (MOP) o Azure Active Directory (Azure AD) o Graph API. 

**In che modo la concessione delle licenze per il Programma CSP per Microsoft Cloud Germania si differenzia dalla concessione delle licenze per CSP Commercial?**

Il Contratto per i rivenditori Microsoft Cloud (MCRA) e il Contratto per i distributori Microsoft Cloud (MCDA) per il Programma CSP per Microsoft Cloud Germania differiscono dall'MCRA Commercial e dall'MCDA Commercial nei due modi seguenti: 
-   Richiedono ai clienti di ricevere il Contratto Microsoft Cloud (MCA) per Cloud Germania con il supplemento ai termini dei servizi online per Cloud Germania 
-   Includono modifiche ai requisiti relativi ai dati dei clienti. L'MCA per Cloud Germania è diverso dall'MCA Commercial per i due punti seguenti: 
-   Include il supplemento ai termini dei servizi online, che include impegni aggiuntivi sulla protezione dei dati di Microsoft per la residenza dei dati, il ruolo del trustee dei dati e l'accesso ai dati dei clienti 
-   Include il Contratto del trustee dei dati. L'Allegato 1 del supplemento contiene un Contratto trustee dei dati-cliente, mediante il quale T-Systems si impegna direttamente nei confronti del cliente in merito al controllo e all'accesso ai dati dei clienti 

**Quali sono i requisiti per la vendita del Programma CSP per Microsoft Cloud Germania?**

I criteri di qualificazione per la vendita del Programma CSP per Microsoft Cloud Germania sono identici ai criteri di qualificazione per CSP Commercial, aperti sia a partner diretti sia a rivenditori indiretti. Non è necessario essere rinominato come rivenditore indiretto per il Programma CSP per Microsoft Cloud Germania se si è approvati come rivenditore indiretto per il Programma CSP in Commercial. 

**Sono un partner esistente, cosa devo fare per vendere il Programma CSP per Microsoft Cloud Germania?**

I partner dovranno eseguire l'onboarding al Programma CSP per Microsoft Cloud Germania. (L'onboarding per Microsoft Cloud Germania è separato dal processo di onboarding per CSP Commercial). Una volta eseguito l'onboarding, è possibile creare clienti ed effettuare transazioni su offerte disponibili all'interno di Microsoft Cloud Germania. Se utilizzi le API, consulta le informazioni aggiuntive riportate di seguito sulle API. 

**Non sono ancora un partner CSP. Dove posso trovare ulteriori informazioni su come diventare un partner CSP?**

Visita il portale MPN per ulteriori informazioni sui modelli di registrazione disponibili per i partner. 

**I prezzi per i clienti per il Programma CSP per Microsoft Cloud Germania sono uguali a quelli per CSP Commercial?**

No. I prezzi riflettono i costi aggiuntivi necessari per fornire il valore unico di Microsoft Cloud Germania. 

**Il modello di sconto per i partner per il Programma CSP per Microsoft Cloud Germania è uguale a quello per CSP Commercial?**

Sì. Il modello di sconto per i partner per il Programma CSP per Microsoft Cloud Germania è uguale a quello per CSP Commercial. 

**Ci sono differenze negli incentivi per i partner per il Programma CSP per Microsoft Cloud Germania rispetto a CSP Commercial?**

No. Gli incentivi per i partner per il Programma CSP per Microsoft Cloud Germania sono gli stessi incentivi per i partner per CSP Commercial. Per ulteriori informazioni sugli incentivi per i partner, visita il portale MPN per le informazioni più recenti. 

**Sono state apportate modifiche alla fatturazione per il Programma CSP per Microsoft Cloud Germania rispetto a CSP Commercial?**
No. I criteri e i requisiti di fatturazione per il Programma CSP per Microsoft Cloud Germania sono gli stessi di CSP Commercial. Per ulteriori informazioni sulla fatturazione, consulta la sezione Risorse nel portale MPN. 

**I requisiti di supporto per il Programma CSP per Microsoft Cloud Germania sono uguali a quelli per CSP Commercial?**

Sì. Per ulteriori informazioni sul supporto, consulta la sezione Risorse nel portale MPN. 

**A quali segmenti di clienti posso vendere?**

Dopo essere stato approvato come partner per MCD CSP, puoi vendere ai clienti commerciali in UE, EFTA e Regno Unito. 

**Quali prodotti/servizi saranno disponibili nel Programma CSP per Microsoft Cloud Germania?**

All'avvio, saranno disponibili Microsoft Office 365 Germany e Microsoft Azure Germany. Microsoft Dynamics 365 Germany (CRMOL) è pianificato per il secondo trimestre 2017. Aggiungeremo altri servizi non appena saranno disponibili.

Al momento sono disponibili Infrastructure as a service (IaaS), piattaforma delle fondamenta e offerte dati. Il piano viene aggiornato regolarmente e forniremo della documentazione, come la documentazione per CSP Commercial nel Centro per i partner (autenticazione richiesta).

**Per i dati ospitati in centri dati Microsoft a Dublino o Amsterdam, è possibile passare a Microsoft Cloud Germania?**

Per facilitare e mantenere l'integrità dell'isolamento di Microsoft Cloud Germania, non sono disponibili meccanismi per spostare un ambiente esistente Microsoft Cloud di un cliente in Microsoft Cloud Germania. Se un cliente/partner dispone di un ambiente online Office 365, Azure o Dynamics CRM, sarà responsabile di eventuali spostamenti dei dati nel nuovo ambiente all'interno di Microsoft Cloud Germania.
